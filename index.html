Comment les professionnels du numérique font pour capter et retenir l'attention des utilisateurs, et comment faire en sorte de mieux contrôler la récupération et le traitement des données à des fins commerciales

Le but de ce site est de vous expliquer quelles techniques sont utilisées pour que vous visitiez (et restiez le plus longtemps possible) les sites internet, les méthodes utilisées pour collecter des données qui peuvent ensuite être utilisées pour cilbler vos centres d'intérêt, ainsi que vous communiquer quelques informations pour vous protéger contre ces "pièges"

La captation d'attention

Comment on capte l'attention avec les notifications

  • Réseaux sociaux (son de notif tout le long où il est sur la page, avec message à la fin "les sons ne vous ont pas perturbé?"
  • Popup de pub / offre promotionnelle, etc (afficher popup après quelques secondes? ou quand il veut sortir?)
  • Popup de confirmation pour quitter la page (afficher popup confirmation quand il quitte)

Comment on capte l'attention grâce au design

  • Design captif (F)
  • Dark design / dark pattern (fait de “masquer” certaines infos, genre inclure automatiquement une assurance dans le panier, inverser annuler/valider)

Comment on capte l'attention avec les techniques de captation

  • FOMO (Fear Of Missing Out)
  • Scroll infini
  • Retargeting (sites visités)
  • Suggestions automatiques (pub ciblées)

Les données

Les cookies

Comment on récupère les informations grâce aux cookies

Comment se protéger contre la récupération de données via les cookies

  • Refuser les cookies non essentiels
  • navigation en mode privé
  • Saisir l'URL directement plutôt qu'utiliser un moteur de recherche (données de recherche = info)

Les données GPS

Comment on récupère les informations grâce aux données gps

Comment se protéger contre la récupération de données via le gps

  • désactiver récupération données google (voir lien)
  • Désactiver données et géolocalisation dés qu'on s'en sert pas

Les formulaires

Comment on récupère les informations grâce aux formulaires (cases à cocher, etc)

Comment se protéger contre la récupération des données via les formulaires

  • Vérifier les cases cochées ou non (transmission des données à fins commerciales)
