La captation d'attention

Comment on capte l'attention avec les notifications

  • Réseaux sociaux (son de notif tout le long où il est sur la page, avec message à la fin "les sons ne vous ont pas perturbé?"
  • Popup de pub / offre promotionnelle, etc (afficher popup après quelques secondes? ou quand il veut sortir?)
  • Popup de confirmation pour quitter la page (afficher popup confirmation quand il quitte)

Comment on capte l'attention grâce au design

Design captif (F)
Dark design / dark pattern (fait de “masquer” certaines infos, genre inclure automatiquement une assurance dans le panier, inverser annuler/valider)
